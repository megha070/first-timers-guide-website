D3.js, What!

D3.js (Data-Driven Documents) is a JavaScript library for producing dynamic, interactive data visualizations in web browsers. D3.js API contains various methods which can be grouped into the following logical units - Selections, Transitions, Arrays, Math, Color, Scales, SVG, Time, Layouts, Geography, Geometry, Behaviors.


D3.js helps you visualize data by combining powerful visualization and interaction techniques with a data-driven approach to DOM manipulation. It also provides full capabilities of modern browsers by designing the right visual interface for your data.
D3.js version 1.0.0 was released back in 18 February 2011.
